# Introduction to Robotics

The course aims to teach basic concepts and key algorithms of robotics. According to the course, students can acquire key techniques and design methods of an autonomous robot, and learn about the advanced robotic techniques. The course will cover structures, kinematics, dynamics, trajectory planning, simulation tools, mobile robotics, sensing, and robotic control. The course is of high comprehension, with emphasis on cultivating and endowing students the capabilities of designing, analyzing and applying the learned knowledge on robotics.
